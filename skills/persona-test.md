# S-04: 페르소나 테스트 (Steam/콘솔 글로벌)

## 목적
Steam/콘솔 글로벌 유저 심리로 시스템을 검증한다.
"이 게임을 사서 할 가치가 있나?" 판단 구조에 집중.
모바일 과금 페르소나는 presets/idle-rpg.md의 BM 섹션 참조.

---

## 핵심 페르소나 8종

### P1 — 코어 액션 게이머 (Core Action Player)
- **특징**: 소울류, 액션 RPG 수백 시간, 기계적 숙련도가 정체성
- **핵심 동기**: 극복의 쾌감, 빌드 최적화, 정복감
- **구매 트리거**: "도전적이다" 리뷰 + 유튜브 보스 전투 영상
- **이탈 트리거**: "너무 쉽다", "패턴 단조롭다", "빌드 하나뿐", "패리 판정 이상함"
- **Steam 행동**: 상세 리뷰 작성, 도전과제 100% 추구, 공략 유튜브 제작
- **커뮤니티 영향**: 부정 리뷰 파급력 큼 → 얼리 난이도 설계 중요
- **체크**: "최고 난이도가 진짜 어려운가? 숙련자 빌드 공간이 있는가?"

### P2 — 스토리 몰입러 (Story-First Player)
- **특징**: 위쳐, RPG, 어드벤처 중심, 게임을 소설/영화처럼 소비
- **핵심 동기**: 감정적 연결, 캐릭터 애착, 세계관 탐구
- **구매 트리거**: "스토리가 뛰어나다" + 세계관 설명 트레일러 + 스팀 설명글
- **이탈 트리거**: "NPC가 모두 같다", "엔딩이 허무하다", "대화를 강제 스킵 못 함"
- **Steam 행동**: 리뷰에 "스토리 스포 주의" 언급, 팬 아트, 2회차 플레이
- **과금 성향**: DLC 구매율 높음 — 스토리 DLC에 즉각 반응
- **체크**: "엔딩에서 감정적 카타르시스가 있는가? 기억에 남는 NPC가 있는가?"

### P3 — 캐주얼 탐험가 (Casual Explorer)
- **특징**: 데이브더다이버, 스타듀밸리 류, 부담 없이 즐기는 게이머
- **핵심 동기**: 스트레스 해소, 발견의 즐거움, "다음에 뭐가 나올까" 기대
- **구매 트리거**: "힐링된다" + "부담 없다" + Steam 할인 시 충동 구매
- **이탈 트리거**: "게임오버 너무 자주", "조작 복잡", "30분 동안 죽었다"
- **Steam 행동**: 별점은 잘 안 씀, 친구 추천, 스크린샷 공유
- **세션 패턴**: 1~2시간 단위, 주말 집중
- **체크**: "첫 30분이 즐거운가? 사망 패널티가 가벼운가?"

### P4 — 인디 서포터 (Indie Supporter)
- **특징**: 인디 게임 발굴이 취미, 얼리 액세스 자주 구매, 개발자 응원
- **핵심 동기**: 숨은 보석 발견, 독특한 경험, 개발자와의 연결감
- **구매 트리거**: Itch.io/데모 플레이 후, 개발자 SNS 팔로우, 위시리스트 오래 유지
- **이탈 트리거**: "업데이트가 없다", "버그 너무 많다", "개발 방치 느낌", "로드맵 사라짐"
- **Steam 행동**: 위시리스트 → 구매 → 리뷰 → 로드맵 댓글 → 개발자 토론 기여
- **얼리 액세스 핵심 유저층**: 이들이 만족해야 입소문 시작됨
- **체크**: "데모/얼리 액세스가 충분한가? 업데이트 소통 계획이 있는가?"

### P5 — 최적화 빌더 (Min-Maxer)
- **특징**: 패스 오브 엑자일, 루트슈터 빌드 수백 시간, 위키 기여자
- **핵심 동기**: 수치 최적화, 메타 파악, 내 빌드 자랑
- **구매 트리거**: "빌드 다양성 풍부" + 위키/DB 존재 확인
- **이탈 트리거**: "메타 빌드 하나뿐", "밸런스 패치로 내 빌드 망가짐", "수치 불투명"
- **Steam 행동**: 위키 기여, 빌드 가이드 유튜브, 토론 게시판 활발
- **개발자 관점 주의**: 이들이 밸런스 문제를 가장 빠르게 발견함
- **체크**: "서로 다른 10가지 유효한 빌드가 있는가? 수치가 명확히 공개되는가?"

### P6 — 스트리머/콘텐츠 크리에이터 (Content Creator)
- **특징**: 게임을 콘텐츠 소재로 플레이, 시청자 반응 고려
- **핵심 동기**: 재밌는 순간 생성, 시청자와 공유, 바이럴 가능성
- **구매 트리거**: 다른 스트리머가 클립으로 퍼뜨림 → 직접 해봄
- **이탈 트리거**: "매번 같은 경험", "놀라움 없다", "시청자가 지루해한다"
- **Steam 행동**: 트위치/유튜브에서 비공식 마케팅, Clip 공유
- **개발자 관점**: 이들이 움직이면 판매량이 급등함 — 키 배포 전략 대상
- **설계 팁**: 공유 가능한 순간(Sharable Moment)을 의도적으로 설계
  예) 예상 못 한 보스 패턴, 웃긴 물리 버그, 극적인 역전
- **체크**: "예상치 못한 재밌는 순간이 있는가? 버라이어티 런이 가능한가?"

### P7 — 컴플리셔니스트 (Completionist)
- **특징**: 도전과제 100%, 모든 엔딩, 숨겨진 요소 발굴
- **핵심 동기**: 완성의 만족감, 모든 컨텐츠 소비
- **구매 트리거**: "숨겨진 요소 많다" + 도전과제 목록 사전 확인
- **이탈 트리거**: "도전과제가 고통스럽다 (버그/운에 의존)", "2회차 의미 없다"
- **Steam 행동**: 도전과제 가이드 작성, 긴 플레이타임 리뷰, 세이브 백업 꼼꼼히
- **개발자 관점**: 평균 플레이타임 올려주는 핵심 유저
- **체크**: "도전과제가 재미있는 과제인가? 2회차 플레이 동기가 있는가?"

### P8 — 글로벌 비영어권 유저 (International Player)
- **특징**: 일본, 독일, 브라질, 동남아 유저 — 언어 장벽이 구매 결정에 큰 영향
- **핵심 동기**: 자국어 지원 여부, 현지 커뮤니티 존재
- **구매 트리거**: Steam 상세 페이지의 지원 언어 목록 확인
- **이탈 트리거**: "기계 번역 같다", "UI 텍스트만 번역됨 (대화 미번역)"
- **Steam 행동**: 현지화 관련 리뷰, 해당 언어 커뮤니티 공유
- **설계 팁**: 텍스트 최소화 + 비주얼 우선 UI = 현지화 비용 절감
- **현지화 우선순위**: 영어 → 일본어(콘솔 필수) → 중국어 간체 → 한국어
- **체크**: "최소 영, 일, 중(간체) 지원 계획이 있는가?"

## Steam 리뷰 패턴 분석

### 부정 리뷰 유발 Top 10
```
1. "너무 짧다"          — 볼륨 기대치 미충족
2. "버그가 너무 많다"   — QA 부족 (얼리 액세스라도 기본 QA 필수)
3. "난이도 밸런스 이상" — 초반 너무 쉽다가 갑자기 벽
4. "조작감이 나쁘다"    — 느리고 둔함, 입력 지연
5. "반복이 심하다"      — 컨텐츠 다양성 부족
6. "엔딩이 실망스럽다"  — 스토리 게임에서 치명적
7. "가격 대비 내용 적다"— 플레이타임 대비 가격 불균형
8. "세이브 간격 너무 길다" — 긴장감이 스트레스로 전환
9. "현지화가 나쁘다"    — 기계 번역 또는 오역
10. "최적화 안 됐다"   — 프레임 드롭, 로딩 길음
```

### 긍정 리뷰 유발 Top 10
```
1. "조작감이 살아있다"          — 게임패드/키마 모두
2. "숨겨진 요소가 많다"         — 탐험 동기 지속
3. "보스 전투가 기억에 남는다"  — 클리어 후 성취감
4. "스토리가 예상을 뒤집었다"   — 반전/감동
5. "한 번만 더 하게 된다"       — 루프 중독성
6. "개발자가 열정적이다"        — 빠른 패치, 소통
7. "다양한 빌드가 가능하다"     — 리플레이 가치
8. "OST가 훌륭하다"             — 감정 몰입 강화
9. "가격 대비 많은 내용"        — 가성비 느낌
10. "친구에게 추천하고 싶다"    — 소셜 전파
```
