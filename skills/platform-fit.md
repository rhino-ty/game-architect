# S-07: 플랫폼 적합도 분석기 (Platform Fit Analyzer)

## 목적
게임 컨셉/시스템이 어떤 플랫폼에 어울리는지 진단한다.
"이 게임, 스팀으로 팔아야 해 vs 모바일 가챠로 가야 해?"를 구조적으로 판단.

---

## 가중치 기반 스코어링 (100점 만점)

5개 축의 가중치가 다름. 단순 개수 카운팅이 아니라 항목당 점수가 다르다.

```
[BM 호환성]     40점  ← 가장 중요. 이게 틀리면 다 틀림
[세션 구조]     20점
[장르 메커닉]   20점
[소셜 구조]     10점
[그래픽 기대치] 10점
```

---

## 축 1: BM 호환성 (총 40점)

각 질문은 Steam/콘솔 또는 모바일 쪽으로 점수를 배분한다.

| 질문 | Steam/콘솔 | 모바일 | 배점 |
|------|-----------|--------|------|
| 유저가 "한 번 사면 다 즐긴다"는 기대를 갖는 장르인가? | +10 | 0 | 10 |
| 반복 과금(가챠, 에너지, 시즌패스 소멸)이 핵심 수익인가? | 0 | +10 | 10 |
| DLC/확장팩으로 수익 구조 설계가 가능한가? | +8 | 0 | 8 |
| 무료 다운로드 진입이 유저 확보에 필수적인가? | 0 | +7 | 7 |
| FOMO(한정 아이템, 소멸 보상)로 결제를 유도하는 구조인가? | 0 | +5 | 5 |

**축 1 판정 기준:**
- Steam 30+ / 모바일 10- : BM이 Steam에 강하게 맞음
- 모바일 20+ / Steam 15- : BM이 모바일에 강하게 맞음
- 비슷하게 나뉨 : 하이브리드 검토 필요

## 판정 결과 해석

### Steam/콘솔 강적합 (Steam 60+ / 모바일 35-)
```
권장 BM:
  - 프리미엄 가격 ($9.99~$29.99)
  - 스토리 DLC or 확장팩
  - 코스메틱 번들
  - 얼리 액세스 (Roguelite, 오픈월드)

출시 전략:
  - Steam Next Fest 데모
  - 위시리스트 10,000+ 목표 후 출시
  - 스트리머/유튜버 키 배포

대표 사례: 하데스, 홀로우나이트, 데이브더다이버, 스타듀밸리
```

### 모바일 강적합 (모바일 60+ / Steam 35-)
```
권장 BM:
  - 무료 다운로드 + IAP
  - 에너지 시스템 + 가챠 (Pity 포함)
  - 배틀패스/시즌패스
  - VIP 시스템

리텐션 전략:
  - 일일 접속 보상
  - 길드/PvP
  - 시즌 컨텐츠

→ presets/idle-rpg.md 또는 presets/card-game.md 참조
```

### 하이브리드 가능 (양쪽 점수 비슷)
```
BM을 플랫폼별로 분리 설계:
  PC 버전:    무료 플레이 or 프리미엄 (입문 장벽 낮춤)
  모바일:     에너지/가챠 (주 수익원)

원신 모델: PC는 유저 수, 모바일이 수익의 대부분

주의: "둘 다 한다"가 "둘 다 애매하다"가 되지 않도록
  → 어느 플랫폼을 Primary로 볼 것인지 먼저 결정
```
