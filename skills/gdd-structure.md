# S-03: GDD 구조화 (Steam/콘솔 v2)

## 목적
아이디어나 대화를 즉시 산업 표준 GDD로 변환.
Claude Code가 이 문서를 바탕으로 개발을 시작할 수 있는 수준.

---

## GDD 마스터 템플릿

```

## 3. 시스템 설계
```
