# S-09: 오디오 설계 (Audio Design)

## 목적
"OST가 훌륭하다"는 Steam 긍정 리뷰 Top 10 중 하나.
1인 개발자가 오디오를 어떻게 확보하고, 구조화하고, 구현하는지 전방위 커버.

---

## 오디오의 게임 내 역할

```
BGM:  분위기 설정 + 감정 유도 + 장소/상황 식별
SFX:  행동 피드백 + 몰입 강화 + Game Juice의 핵심
보이스: 캐릭터 생동감 (없어도 되지만 있으면 레벨 업)
앰비언트: 세계의 살아있음 (바람소리, 새소리, 군중)
```

---

## Part 1: BGM 전략

### 1인 개발자의 현실적 BGM 확보 경로

```
옵션 A: 직접 제작
  도구: LMMS (무료), GarageBand (Mac 무료), FL Studio ($99+)
  난이도: 높음 — 음악 이론 기초 필요
  장점: 비용 0, 저작권 완전 소유, 수정 자유
  현실: 게임 개발하면서 음악도 만들기는 매우 힘듦

옵션 B: 프리랜서 작곡가 의뢰 (권장)
  가격: 1분당 $50~$300 (퀄리티에 따라)
  찾는 곳: itch.io 작곡가 섹션, Twitter #gameaudio,
            Fiverr, MusicForGames.net
  계약 시 반드시 확인:
    □ 상업적 사용권 포함 여부
    □ 독점 vs 비독점 라이센스
    □ Steam/콘솔 배포 포함 여부
    □ 크레딧 표기 방식

옵션 C: 라이센스 구매 (빠른 개발용)
  무료: OpenGameArt.org, FreeMusicArchive, ccMixter
  유료: Epidemic Sound ($15/월), Artlist ($199/년)
  ※ 상업 게임은 반드시 상업 라이센스 확인
  ※ Steam 출시 = 상업적 사용 → 무료 CC BY-SA도 조건 확인 필요

옵션 D: AI 생성 (실험적)
  도구: Suno, Udio, MusicGen
  현황 2024: 퀄리티 급상승 중, 저작권 논란 존재
  실용적 사용법: 프로토타입 단계 임시 BGM으로 사용,
                출시 전 교체하는 것이 안전
```

### BGM 트랙 구성 (최소 필요량)

```
BGM_MINIMUM = {
    '메인 메뉴': 1,          # 게임의 첫인상
    '일반 탐험': 2~3,        # 구역/분위기별
    '전투': 1~2,             # 일반 전투 + 보스 전투
    '보스': 1~2,             # 긴장감 있는 별도 테마
    '승리/클리어': 1,        # 짧은 스팅 (15~30초)
    '게임오버': 1,           # 짧은 스팅
    '엔딩': 1,               # 감동적인 마무리
}
# 총 최소: 8~12트랙
# 권장: 15~20트랙 (구역별 변주 포함)
```

### 적응형 BGM (Adaptive Music) 설계

```
레벨 1 (단순): 상황별 다른 트랙 재생
  탐험 중 → 탐험 BGM
  적 발견 → 전투 BGM (페이드 전환 2초)
  적 처치 → 탐험 BGM 복귀

레벨 2 (중간): 레이어 기반 적응형
  베이스 트랙 항상 재생
  + 전투 레이어 (타악기) 전투 시 추가
  + 긴장 레이어 (낮은 현악) HP 낮을 때 추가
  도구: FMOD Studio (무료 티어 있음), Wwise (소규모 무료)

레벨 3 (고급): 절차적 음악 생성
  1인 개발에는 과함 → 레벨 2로 충분
```

## Part 2: SFX (Sound Effects) 전략

### SFX 확보 경로

```
무료 소스:
  freesound.org   — 커뮤니티 기반, 라이센스 각각 확인 필요
  OpenGameArt.org — 게임 전용, 대부분 상업 사용 가능
  Zapsplat.com    — 무료 계정으로 상업 사용 가능
  Kenney.nl       — CC0 (완전 무료), 게임용 팩 다수

직접 제작 (추천):
  BFXR / SFXR     — 레트로 8bit SFX 생성기, 무료, 즉시 사용
  ChipTone        — 더 다양한 레트로 SFX, 무료
  Leshy SFMaker   — 브라우저 기반, 무료

프리랜서 의뢰:
  간단한 SFX 팩: $50~$200
  복잡한 게임 전체 SFX: $500~$3,000
```

### 필수 SFX 목록 (액션 게임 기준)

```
전투:
  □ 공격 휘두름 (무기 유형별)
  □ 적 피격 (일반 / 치명타 / 방어)
  □ 플레이어 피격
  □ 적 사망
  □ 플레이어 사망

이동:
  □ 발소리 (지형별: 돌 / 흙 / 물 / 금속)
  □ 점프
  □ 착지 (높이별 3단계)
  □ 대쉬

UI:
  □ 메뉴 hover
  □ 메뉴 select / confirm
  □ 메뉴 cancel / back
  □ 아이템 획득
  □ 레벨업 / 업그레이드

환경:
  □ 문 열림
  □ 보물상자 열림
  □ 스위치 / 트리거
  □ 앰비언트 (구역별)
```

### SFX 레이어링 기법

```
단순 SFX vs 레이어드 SFX 비교:

단순: 공격 버튼 → 검 휘두름 SFX 1개 재생
레이어드:
  공격 버튼
  → 바람 가르는 소리 (공격 시작, 즉시)
  → 무기 임팩트 (타격 순간, 히트스톱과 동기화)
  → 잔향 / 에코 (0.1~0.2초 후)
  [선택] → 캐릭터 기합 음성

결과: 같은 행동이지만 "살아있는 느낌" 3배

구현:
  Unity: AudioMixer + 여러 AudioSource 레이어
  Godot: AudioStreamPlayer + AudioBus
```

## Part 3: 오디오 엔진 선택

### 기본 엔진 오디오 vs 미들웨어

```
기본 엔진 오디오:
  Unity AudioMixer / Godot AudioBus
  장점: 추가 비용 없음, 간단한 게임에 충분
  단점: 적응형 음악, 복잡한 믹싱에 한계

FMOD Studio (권장 미들웨어):
  가격: 수익 $200K 이하 무료
  통합: Unity/Godot 모두 공식 지원
  기능: 적응형 음악, 실시간 파라미터, 강력한 믹서
  학습: 중간 수준 (공식 튜토리얼 잘 되어 있음)

Wwise (대안):
  가격: 인디 플랜 무료 (제한 있음)
  특징: 업계 표준, AAA에서도 사용
  단점: FMOD보다 복잡함

1인 개발 권장:
  단순한 게임: 기본 엔진 오디오
  Roguelite / 오픈월드: FMOD Studio
```

## Part 4: 오디오 믹싱 기초

### 볼륨 레벨 기준 (dB)

```
마스터 버스:      0 dB (기준점)
BGM:            -10 ~ -6 dB (배경이므로 조금 낮게)
SFX:             -6 ~ -3 dB (BGM보다 선명하게)
UI SFX:         -12 ~ -8 dB (가장 작게)
보이스:          -4 ~ -1 dB (가장 선명하게)
앰비언트:        -18 ~ -12 dB (매우 작게)

전투 시 자동 덕킹 (Ducking):
  전투 SFX가 크게 날 때 BGM을 -4~-6 dB 자동 감소
  FMOD/Wwise의 Sidechain Compressor로 구현
```

### 공간감 설계 (3D 게임)

```
3D 공간 오디오:
  거리 감쇠: 20m 기준 -6dB, 40m에서 -12dB (로그 곡선)
  방향감: 패닝으로 좌/우 구분
  환경 반향: 실내(많음) vs 야외(적음)

실내/실외 전환:
  실외: 리버브 적음, 하이 컷 없음
  실내: 리버브 많음, 약간의 하이 컷 (먹먹한 느낌)
  동굴: 딜레이 에코 + 강한 리버브

Unity AudioMixer:
  Reverb Zone 컴포넌트로 구역별 자동 적용
Godot:
  AudioStreamPlayer3D + AudioBusLayout로 구현
```

## Part 5: 오디오 파일 스펙

```
BGM:
  형식: OGG Vorbis (Steam/대부분 플랫폼 최적)
  비트레이트: 128~192 kbps (품질 vs 용량 균형)
  샘플레이트: 44,100 Hz
  루프: 루프 포인트 정밀하게 설정 (뚝 끊기지 않게)

SFX:
  형식: WAV (편집용) → OGG (실제 배포용)
  비트레이트: 256 kbps+ (짧지만 선명해야 함)
  샘플레이트: 44,100 Hz

모바일 최적화 (Switch/Mobile 포팅 시):
  BGM: 112 kbps OGG
  SFX: 64~96 kbps OGG
  → 용량 크게 절감, 체감 품질 변화 거의 없음
```

텍스트 RPG / VN:

## 오디오 개발 체크리스트

```
프로토타입 단계:
  □ 임시 BGM 설정 (개발 분위기 유지용)
  □ 핵심 SFX 3~5개 (공격, 피격, 점프)

알파 단계:
  □ BGM 전체 트랙 확보 (최종본 아니어도 됨)
  □ 전체 SFX 목록 작성 + 60% 이상 완성
  □ 볼륨 레벨 1차 밸런싱

베타 / 출시 전:
  □ 모든 SFX 완성
  □ 적응형 음악 구현 (전투/탐험 전환)
  □ 전체 볼륨 믹싱 완료
  □ 옵션 메뉴: BGM/SFX/마스터 볼륨 분리 조절
  □ 헤드폰 + 스피커 + TV 스피커 3환경 테스트
```
