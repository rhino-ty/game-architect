## 글로벌 현지화 우선순위

```
필수 (매출 영향 큰 순):
1. 영어 (기본)
2. 중국어 간체 (Steam 최대 유저)
3. 일본어 (콘솔 포팅 필수)
4. 한국어 (국산 게임 이점)
5. 독일어 / 프랑스어 (유럽)
6. 포르투갈어-브라질 (빠르게 성장)
7. 러시아어
8. 스페인어

현실적 예산 배분:
- 텍스트 번역: 언어당 $500~$3,000 (분량에 따라)
- 음성 녹음: 텍스트 비용의 5~10배
- 전략: 텍스트 번역 먼저, 음성은 영어만 or 히트 후 추가
```

## 인디 개발 현실 예산 참고

```
솔로 개발 (1년):
  생활비 제외 개발 비용: $0 ~ $5,000 (에셋, 툴, 사운드)
  Steam 등록: $100
  마케팅 (최소): $500~$2,000

소규모 팀 3인 (2년):
  에셋/아웃소싱: $10,000~$30,000
  QA/로컬라이제이션: $5,000~$15,000
  마케팅: $5,000~$20,000

손익분기점 계산:
  판매 단가 $19.99 × Steam 수수료(30%) = 수입 $14
  개발 비용 $30,000 ÷ $14 = 2,143 판매량에서 손익분기
```
