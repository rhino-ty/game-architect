```

  .gitignore에 반드시 포함:
    .import/
    export_presets.cfg
    *.translation

Unity 프로젝트 Git 설정:
  .gitignore에 반드시 포함:
    Library/
    Temp/
    Logs/
    .vs/
```

## 프레스 키트 (presskit 구성)

```
presskit() 사이트 (dopresskit.com)에서 자동 생성 가능.

필수 포함 내용:
  □ 게임 제목 + 태그라인
  □ 한 줄 설명 + 상세 설명
  □ 스크린샷 10장+ (고해상도)
  □ 트레일러 링크
  □ 로고 (흰 배경 + 투명 배경 모두)
  □ GIF 3~5개 (게임플레이 하이라이트)
  □ 개발자 정보 + 연락처
  □ Steam/itch.io 링크
  □ 가격 + 출시일
  □ 지원 플랫폼 + 언어

프레스 키트 URL:
  yourwebsite.com/presskit or
  dopresskit.com/[게임명]
```

## 1인 개발 하드웨어 최소 사양 (참고)

```
개발 PC:
  RAM: 16GB (Unity/Godot 개발 최소), 32GB 권장
  GPU: 중간급 이상 (테스트용, 최소 사양 게임은 낮은 GPU로도 가능)
  저장: SSD 512GB+ (프로젝트 + 에셋 용량)

모바일 테스트 (iOS):
  Mac 필요 (Xcode 빌드 필수)
  iPhone 구형 기기로 최저 사양 테스트

콘솔 개발 키트:
  Nintendo Switch: 개발 계정 신청 필요 (닌텐도에 직접)
  PlayStation: PS 파트너 포털 신청
  → 개발 키트 대여 비용 발생 (수백만원대)
  → 1인 개발 시 PC/Steam 먼저, 콘솔 포팅은 나중에
```
