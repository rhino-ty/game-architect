# Preset: 액션 RPG (Action RPG)

## 장르 정의
실시간 액션 전투 + RPG 성장의 결합. 조작 숙련도와 빌드 최적화가 동시에 중요.
핵심: **전투의 쾌감** + **캐릭터 커스터마이징** + **빌드 다양성**

---

## 핵심 루프

```
[지역 탐험] → [적과 전투 (실시간)] → [스킬/회피/콤보] → [보스 처치] → [장비/스킬 획득] → [성장] → [더 강한 구역]
```

---

## 필수 시스템 체크리스트

### 전투 핵심
- [ ] **히트스톱**: 타격감 핵심 (권장 0.05~0.1초 프리즈)
- [ ] **무적 프레임(I-Frame)**: 회피 시 무적 구간 (권장 0.2~0.4초)
- [ ] **스태거 시스템**: 적이 맞으면 경직되는 구간
- [ ] **파리 시스템**: 완벽 타이밍 패리 → 반격 기회
- [ ] **콤보 시스템**: 공격 연계 시 추가 효과

### 스킬 시스템
- [ ] **스킬 슬롯 수**: 4~8개 권장 (조작 복잡도 균형)
- [ ] **쿨타임 관리**: 리소스 (MP/스태미나) or 쿨타임 순수
- [ ] **스킬 시너지**: 스킬 A 사용 후 B가 강화됨 등

### 장비 시스템
- [ ] **세트 효과**: N개 장착 시 추가 효과
- [ ] **고유 아이템**: 메커닉이 변하는 전설 등급
- [ ] **보조 옵션 (접두/접미)**: 랜덤 옵션으로 빌드 다양성

## 수치 템플릿

### 전투 느낌 수치
```python
COMBAT_FEEL = {
    'hitstop_duration': 0.07,      # 초 (타격감)
    'iframe_duration': 0.3,        # 초 (회피 무적)
    'stagger_threshold': 0.3,      # 적 HP 30% 이하 or 큰 타격 시 스태거
    'parry_window': 0.15,          # 초 (패리 타이밍 윈도우)
    'combo_timeout': 0.8,          # 초 (콤보 입력 허용 시간)
    'knockback_distance': 1.5,     # 미터
}
```

### 보스 패턴 설계
```
페이즈 1 (HP 100~60%): 기본 패턴 2~3종
페이즈 2 (HP 60~30%): 신규 패턴 추가 + 공속 15% 상승
페이즈 3 (HP 30~0%):  격분 + 신규 특수 패턴 + 데미지 20% 상승
```

---

## Edge Case (장르 특수)

1. **무적 프레임 남용**: 모든 공격을 회피 스팸으로 무력화 → 패리 보상 > 회피 보상
2. **원거리 치즈**: 원거리 빌드가 근접 불필요하게 함 → 원거리 패널티 or 근접 딜 우위
3. **무한 CC 루프**: 스킬 연계로 적이 영원히 넘어짐 → CC 면역 타이머 (tenacity 시스템)
4. **보스 DPS 경쟁**: 다수 플레이어 시 탱커/힐러 역할 불필요 → 오라/버프 메커닉으로 역할 분화

## 레퍼런스 게임
- **다크소울/엘든링**: 패리 & 스태거 시스템 정석
- **원신**: 속성 반응 + 파티 조합 시너지
- **패스 오브 엑자일**: 극단적 빌드 다양성
- **몬스터헌터**: 무기 유형별 전투 스타일 차별화
