# Preset: 오픈월드 RPG (Open World RPG)

## 장르 정의
탐험 자유도가 핵심. 거대한 세계를 돌아다니며 발견, 전투, 성장, 스토리를 경험.
핵심: **발견의 희열** + **나만의 캐릭터 성장**

---

## 핵심 루프

```
[탐험] → [발견(던전/NPC/아이템)] → [전투] → [보상] → [성장] → [새 구역 탐험]
    ↑___________________________________________________|
```

### 보조 루프
```
[메인 퀘스트] ↔ [사이드 퀘스트] ↔ [수집 요소] ↔ [제작 시스템]
```

---

## 필수 시스템 체크리스트

### 월드 설계
- [ ] **구역 단계화**: 레벨 권장 구역 vs 자유 탐험 균형
- [ ] **POI(Point of Interest) 밀도**: 화면 이동 2~3분마다 발견 요소
- [ ] **레벨 스케일링**: 지역 고정 vs 플레이어 추종형 결정
- [ ] **패스트 트래블**: 발견 후 해금 방식 (텔레포트 스톤, 마차 등)

### 전투 시스템
- [ ] **전투 깊이**: 액션(스킬샷) vs 턴제 vs 반자동
- [ ] **속성 상성 시스템**: 불/물/번개 등 삼각관계
- [ ] **CC 면역**: 보스는 단순 CC 무효, 약점 부위 공격으로 교환

### 캐릭터 성장
- [ ] **클래스/직업 시스템**: 고정 vs 자유 빌드
- [ ] **스킬 트리**: 너비 우선 (다양성) vs 깊이 우선 (특화)
- [ ] **장비 티어**: 화이트-그린-블루-퍼플-오렌지 5단계 권장

## 수치 템플릿

### 레벨별 스탯 스케일링
```python
# 오픈월드 RPG 레벨 스탯 공식 (50레벨 기준)
def player_stat(level, base=100, growth=0.08):
    """S커브 기반 성장 - 초반 빠름, 후반 완만"""
    import math
    return base * (1 + growth) ** level

def enemy_scaling(player_level, zone_level, elite=False):
    """
    zone_level: 해당 구역 권장 레벨
    플레이어가 zone_level + 5 이상이면 약한 적, -5 이하면 강함
    """
    level_diff = player_level - zone_level
    multiplier = 1.0 / (1 + 0.1 * level_diff)  # 레벨 차이 완화
    base_hp = 500 * (1.15 ** zone_level)
    return base_hp * multiplier * (2.5 if elite else 1.0)
```

### 퀘스트 보상 밸런싱
```
메인 퀘스트 XP = 현재 레벨 달성 필요 XP × 0.15
사이드 퀘스트 XP = 메인 퀘스트 XP × 0.40
일일 퀘스트 XP = 메인 퀘스트 XP × 0.20 (반복)

장비 드롭 기준: 적 레벨 ± 2 범위 장비
```

## 오픈월드 특유 Edge Case

1. **레벨 점프 치즈**: 고레벨 구역 몬스터를 지형으로 처치 → 의도 여부 결정
2. **경제 붕괴**: 파밍 구역 최적화 → 재화 무한 생성 → 인플레 → Sink 강화 필요
3. **메인 퀘스트 스킵**: 모든 것을 먼저 탐험하고 메인 퀘스트 방치 → 스토리 연결 동기 부여
4. **세계 밀도 문제**: 맵이 크지만 빈 공간 → 랜덤 이벤트 / 앰비언트 이벤트로 채움
5. **패스트 트래블 남용**: 이동 경험을 스킵 → 탐험 경로상 보상 배치로 유인
