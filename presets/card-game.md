# Preset: 카드 게임 (Collectible/Deck-Building)

## 장르 정의
카드 수집과 덱 구성이 핵심. TCG(Trading Card Game), DCG(Digital Card Game), 덱빌딩 Roguelite 포함.
핵심: **수집의 완성도** + **전략적 조합** + **정보 비대칭의 긴장감**

---

## 핵심 루프

### TCG/DCG 형태
```
[카드 획득(가챠/구매)] → [덱 구성] → [대전] → [랭킹/보상] → [카드 획득]
```

### 덱빌딩 Roguelite 형태
```
[런 시작(기본 덱)] → [전투] → [보상 카드 선택] → [전투] → [보스] → [재도전]
```

---

## 필수 시스템 체크리스트

### 카드 시스템
- [ ] **카드 분류**: 유닛/주문/무기/필드 등 타입 체계
- [ ] **코스트 시스템**: 마나/에너지 커브 설계 (2~4코 카드가 핵심)
- [ ] **희귀도 계층**: 일반/희귀/에픽/전설, 전설 덱 제한 (최대 2장)
- [ ] **밸런스 안전장치**: 카드 너프 정책 + 카드 환불 시스템

### 덱 구성 규칙
- [ ] **덱 사이즈**: 표준 20~40장 (장르에 따라)
- [ ] **중복 제한**: 같은 카드 최대 2~3장
- [ ] **색(Color)/속성 제한**: 다색 패널티 or 순수색 보너스
- [ ] **금지/제한 목록**: 경쟁 메타 관리용

### BM 설계 (카드게임 특화)
- [ ] **카드팩 가격**: 1팩 = 1,500원 기준 (5장 구성)
- [ ] **천장(Pity)**: 40팩 전설 보장
- [ ] **와일드카드**: N장마다 원하는 카드 교환권
- [ ] **시즌 로테이션**: 구형 카드 표준 모드 퇴장 주기

### 카드 파워 레벨 공식
```
카드 가치 = (직접 스탯 가치) + (효과 가치) - (코스트 패널티)

예) 2코스트 2/2 바닐라 유닛: 기준값 1.0
    2코스트 2/2 + 드로우 1장: ~1.4 (드로우 가치 0.4)
    2코스트 2/2 + 즉시 피해 2: ~1.3
    2코스트 3/3: ~1.2 (스탯 과잉)
```

### 가챠 시뮬레이션 (카드팩)
```python
import random
from collections import Counter

def open_packs(n_packs, rates={'일반':0.70,'희귀':0.20,'에픽':0.08,'전설':0.02}):
    results = Counter()
    pity = 0
    for _ in range(n_packs):
        for _ in range(5):  # 팩당 5장
            pity += 1
            if pity >= 40:
                results['전설'] += 1
                pity = 0
                continue
            roll = random.random()
            cumulative = 0
            for rarity, rate in rates.items():
                cumulative += rate
                if roll < cumulative:
                    results[rarity] += 1
                    break
    return dict(results)

result = open_packs(100)
print(f"100팩 결과: {result}")
print(f"전설 획득: {result.get('전설', 0)}장")
```

## Edge Case (장르 특수)

1. **파워 크리프**: 신규 카드가 구형보다 항상 강함 → 성능 스펙 문서화 + 내부 레이팅
2. **단일 메타 지배**: 한 덱이 모든 것을 이김 → 메타게임 지표 모니터링 (승률 55% 초과 시 조치)
3. **Pay-to-Win 논란**: 전설 카드 없이 경쟁 불가 → 무료 전설 루트 필수
4. **카드 조합 버그**: 예상치 못한 콤보로 즉사 가능 → 출시 전 조합 그래프 분석
