# Preset: 액션 어드벤처 (Action Adventure)

## 레퍼런스
하데스, 갓오브워, 젤다 시리즈, 홀로우나이트, 데드셀즈

## 장르 정의
실시간 전투 + 탐험 + 내러티브의 삼각 균형.
"전투가 재밌고, 탐험이 의미있고, 캐릭터가 기억에 남는다."

---

## 핵심 루프

```
[탐험] → [전투 조우] → [스킬/회피/조합 실행] → [적 처치] → [보상 + 서사 진전]
    ↑___________________________________________________|
    └─ [보스 처치] → [새 구역 해금] → [서사 전환점]
```

### 서브 루프 (하데스 모델)
```
[런 시작] → [전투 클리어] → [Boon 선택] → [관계 대화] → [사망 or 클리어]
    ↓ (귀환)
[허브 공간에서 서사 진행] → [다음 런] → ...
```

### 무기/스킬 시스템 (하데스 모델)
```
무기 유형: 근접 / 원거리 / 혼합 각 1~3종
각 무기 = 기본 공격 + 특수 공격 + 대쉬 공격
Boon (축복) = 외부에서 부여되는 스킬 변형
  → 기본 공격에 화염 부여
  → 대쉬 후 폭발 발생
  → 피격 시 반격 등

시너지 구성: 2~3개 Boon 조합 시 강력한 빌드 출현 (의도적 설계)
```

### 허브 공간 (Hub World) 설계
```
역할: 서사 진행 + 관계 발전 + 메타 업그레이드 거점

필수 요소:
- 반복 방문 시 대화가 바뀌는 NPC (런 마다 새 대사)
- 메타 성장 UI (영구 스탯 또는 해금)
- 비밀 발견 요소 (숨겨진 방, 숨겨진 대화)

런 사이 서사 진행 트리거:
  - 특정 횟수 사망 후 NPC 대사 변경
  - 특정 아이템 획득 후 이벤트 발생
  - 클리어 횟수에 따른 엔딩 분기
```

## 보스 설계 (갓오브워 모델)

### 보스 품질 기준
```
보스 1체 = 게임 내 가장 집약된 경험

필수 요소:
1. 명확한 시각적 페이즈 전환 (체력 60%, 30% 등)
2. 공격 패턴 학습 → 숙달의 만족감
3. 약점 또는 창의적 돌파 방법
4. 서사적 의미 (왜 이 보스를 처치하는가)
5. 고유 사운드 테마

데이터:
페이즈 수:        2~3 (인디) / 3~4 (AA)
공격 패턴 수:     6~10 (페이즈당)
전투 시간 목표:   5~12분 (첫 클리어 기준)
```

### 보스 스탯 공식
```python
def boss_stats(player_dps, target_fight_time_sec, phase_count=2):
    """
    player_dps: 플레이어 평균 DPS
    target_fight_time_sec: 목표 전투 시간 (초)
    """
    total_hp = player_dps * target_fight_time_sec
    phase_thresholds = [1.0 - (i / phase_count) for i in range(1, phase_count)]
    
    return {
        'total_hp': int(total_hp),
        'phase_thresholds': [f"{t*100:.0f}%" for t in phase_thresholds],
        'recommended_dmg_per_phase': int(total_hp / phase_count),
        # 플레이어 HP 대비 보스 데미지 조정
        'boss_hit_dmg_normal': f"player_maxhp × 0.08",
        'boss_hit_dmg_heavy':  f"player_maxhp × 0.18",
        'boss_hit_dmg_oneshot_warning': f"player_maxhp × 0.35 이하 권장",
    }

# 예: DPS 150, 목표 8분 전투
print(boss_stats(150, 480, 3))
```

## Edge Case (장르 특수)

1. **패리 진입 장벽**: 패리 윈도우가 너무 좁으면 캐주얼 이탈 → 접근성 설정 옵션 필수
2. **Boon 밸런스**: 특정 조합만 강함 → 모든 조합이 "나름대로 재밌음" 보장
3. **사망 패널티**: 너무 가혹하면 좌절, 없으면 긴장 소멸 → Roguelite식 부분 손실
4. **서사-전투 단절**: 전투만 하다가 스토리 잊음 → 허브 공간에서 자연스러운 서사 소비
5. **보스 원스탯**: 플레이어 스탯만 올리면 어떤 보스든 클리어 → 보스별 전략 필수 설계
