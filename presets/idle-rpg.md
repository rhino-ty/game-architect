# Preset: 방치형 RPG (Idle RPG / AFK RPG)

## 장르 정의
방치 중에도 자동으로 성장하며, 유저가 복귀할 때마다 축적된 보상과 다음 결정을 내리는 장르.
핵심: **복귀 도파민** + **최적화 만족감**

---

## 핵심 루프

```
[복귀] → [오프라인 보상 수령] → [장비/스킬 업그레이드] → [새 구간 도전]
    ↓
[방치 시작] → (시간 경과) → [복귀] → ...
```

### 세컨더리 루프
```
[일일 퀘스트] → [길드 레이드] → [PvP 아레나] → [이벤트 미션]
```

---

## 필수 시스템 체크리스트

### 핵심 수집 루프
- [ ] **오프라인 보상 시간 상한**: 권장 8~12시간 (무과금 기준)
  - 과금 시 상한 연장: 최대 24시간
- [ ] **골드 생산량**: 현재 스테이지 적 처치 시 0.5~2초당 1골드 수준
- [ ] **자동 전투**: 항상 켜져 있음, 스킬 우선순위만 설정

### 성장 단계 설계
```
[스테이지 클리어] → [장비 강화] → [스테이지 클리어] → [초월/각성] → [상위 컨텐츠]
```
- 성장 벽(Wall) 위치: 매 10~20 스테이지마다 의도적 벽
- 벽 돌파 수단: 재화 축적 / 장비 강화 / 스킬 레벨업

### BM (방치형 특화)
- [ ] **방치 시간 구매**: 시간 단축 or 상한 연장 (가장 기본 BM)
- [ ] **VIP 시스템**: 누적 과금액 기반 영구 버프
- [ ] **광고 보상**: 2~3배 보상 (30초 광고), 하루 5~10회 한도
- [ ] **배틀패스**: 30일 기준, F2P와 유료 트랙 병행

## 수치 템플릿

### 오프라인 보상 계산
```
OfflineGold = GoldPerSecond × OfflineSeconds × OfflineEfficiency
OfflineEfficiency = 0.5 (무과금) ~ 1.0 (최고 과금)
OfflineSeconds_cap = 28800 (8h, 무과금) ~ 86400 (24h, 과금)
```

### 성장 배율 (각 성장 단계)
```python
GROWTH_STAGES = {
    '강화':    {'max_level': 20, 'stat_per_level': 0.05},  # +5%/레벨
    '돌파':    {'trigger': [5,10,15,20], 'bonus': 0.20},    # +20% 돌파
    '각성':    {'max_level': 5, 'stat_per_level': 0.15},    # +15%/레벨
    '초월':    {'max_level': 3, 'stat_per_level': 0.50},    # +50%/레벨
}

def total_multiplier(강화_lv, 각성_lv, 초월_lv):
    base = 1.0
    base *= (1 + GROWTH_STAGES['강화']['stat_per_level']) ** 강화_lv
    # 돌파 보너스
    breaks = sum(1 for t in [5,10,15,20] if 강화_lv >= t)
    base *= (1 + GROWTH_STAGES['돌파']['bonus']) ** breaks
    base *= (1 + GROWTH_STAGES['각성']['stat_per_level']) ** 각성_lv
    base *= (1 + GROWTH_STAGES['초월']['stat_per_level']) ** 초월_lv
    return base
```

## 리텐션 설계 (방치형 특화)

| 트리거 | 방법 | 목표 |
|--------|------|------|
| Day 1 | 첫 30분 내 오프라인 보상 체험 | 방치 루프 이해 |
| Day 3 | 길드 가입 유도 | 소셜 훅 |
| Day 7 | 첫 과금 패키지 제안 | 첫 결제 전환 |
| Day 30 | 시즌/이벤트 개막 | 복귀 동기 |

---

## Edge Case (장르 특수)

1. **성장 정체**: 스테이지 벽이 너무 두껍다고 느끼면 이탈 → 보상 분배 세분화
2. **콘텐츠 소진**: 최고 레벨 도달 후 할 것이 없다 → 엔드게임 루프 필수
3. **광고 피로**: 광고를 너무 자주 요구하면 부정 경험 → 일일 횟수 강제 상한
4. **복귀 불안**: 장기 이탈 후 "이미 늦었다" 인식 → 복귀 보상 + 캐치업 시스템
