# Preset: Metroidvania

## 레퍼런스
홀로우나이트, 오리와 눈먼 숲, Axiom Verge, 캐슬바니아 SotN, 블러드스테인드

## 장르 정의
탐험 잠금-해제 구조가 핵심. 새 능력을 얻어야만 갈 수 있는 구역이 열리며,
세계 전체가 하나의 퍼즐처럼 연결되어 있다.
"갔다가 못 갔던 곳으로 돌아오는 순간의 쾌감."

---

## 핵심 루프

```
[탐험] → [막힌 구역 발견] → [새 구역에서 능력 획득]
    → [이전 막힌 구역 해금] → [더 넓은 탐험 가능]
         ↑________________________________________________|
```

### 능력 획득 순서 설계 (Critical Path)
```
능력 A → 구역 1~3 접근 가능
능력 B → 능력 A 필요 + 구역 4~6 접근 가능
능력 C → 능력 A+B 필요 + 숨겨진 구역 접근 가능
보스 → 능력 습득 후에만 약점 노출 (또는 전략 가능)
```

---

## 필수 시스템

### 세계 구조 설계 원칙

```
연결성:
  모든 구역은 최소 2개의 경로로 연결될 것
  막힌 길 ≠ 데드엔드 → 항상 "다른 방향" 존재

구역 당 밀도:
  - 탐험 공간: 구역의 60~70%
  - 전투 구간: 20~30%
  - 퍼즐/환경: 10~20%
  - 미니보스: 구역당 1~2체
  - 메인보스: 구역당 1체 (구역 끝)

지도 시스템:
  - 탐색 전: 안개 처리 or 외곽선만
  - 탐색 후: 방 구조 표시
  - 완전 탐색: 수집 요소 위치 표시 (컴플리셔니스트 훅)
```

### 이동 능력 설계 카탈로그

```
수직 이동:
  더블 점프   — 진입 장벽 낮음, 대부분의 게임에서 초반 획득
  벽 점프     — 수직 이동 + 좁은 공간 전용
  대쉬/충전   — 가로 빠른 이동, 무적 프레임 부여 가능
  글라이드     — 수직 하강 제어, 탐험 맵 변화 줌

환경 상호작용:
  폭탄         — 숨겨진 벽 파괴, 퍼즐 해결
  갈고리/그래플 — 천장/벽 이동, 스윙 역학 추가
  수영          — 수중 구역 해금
  빛/어둠 조작  — 숨겨진 경로/적 가시화

전투 연계:
  공중 대쉬    — 전투+이동 복합, 패턴 피하기에 활용
  다운 스트라이크 — 적 머리 밟기로 점프, 수직 퍼즐
```

### 보스 설계 (Metroidvania 특화)

```
Metroidvania 보스의 특수 역할:
1. 능력 교사 (Ability Teacher)
   — 보스가 먼저 능력을 사용해 보여줌
   — 플레이어가 보스를 처치해 그 능력을 얻음
   예) 보스가 벽 점프로 이동 → 처치 후 벽 점프 획득

2. 리턴 보스 (Return Challenge)
   — 초반에 못 이기는 보스 → 능력 얻고 복귀 → 클리어
   — 성장 체감을 극대화하는 장치

3. 선택 보스 (Optional Boss)
   — 메인 루트 아닌 탐험 보상
   — 강력한 선택적 업그레이드 드롭
```

## Edge Case (장르 특수)

1. **길 잃음**: 어디 가야 할지 모르는 상태 → 힌트 NPC or 지도 마커 옵션
2. **능력 잠금 버그**: 능력 없이 구역 통과 가능한 경로 → 체계적 플래그 관리
3. **역주행 지루함**: 이미 클리어한 구역을 다시 지나가는 것 → 패스트 트래블 포인트
4. **초반 이탈**: 능력 없어서 막힘 → 초반엔 탐험 범위 자연스럽게 안내
5. **완벽주의 마비**: 100% 달성 강박 → 탐험도 UI는 선택적으로 (보여줄지 말지)
