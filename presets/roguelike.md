# Preset: Roguelike / Roguelite

## 장르 정의
- **Roguelike** (순수): 런 간 영구 성장 없음, 완전 초기화 (NetHack, Caves of Qud, DCSS)
- **Roguelite**: 죽어도 메타 성장 남음 → 하데스(거울/어둠), Dead Cells(세포), Vampire Survivors

> ⚠️ 흔한 오해: 하데스는 Roguelite. 런마다 어둠으로 영구 업그레이드 가능. Steam 인기 장르는 대부분 Roguelite.


---

## 핵심 루프

```
[런 시작] → [빌드 구성 (아이템/스킬 선택)] → [전투] → [보상/분기] → [사망 OR 클리어]
    ↓ (사망 시)
[메타 성장] → [런 재시작] → ...
```

---

## 필수 시스템 체크리스트

### 런 내 시스템 (Run-Time)
- [ ] **룸 생성 알고리즘**: BSP / Wave Function Collapse / 프리셋 조합
- [ ] **아이템 풀 설계**: 티어 분류 (Normal/Rare/Epic/Legendary), 저주 아이템
- [ ] **빌드 시너지**: 최소 3가지 강력한 조합 의도적 설계
- [ ] **난이도 스케일링**: 층/구역별 적 스탯 배율 (권장: 1.2x per floor)
- [ ] **희귀도 드롭률**: 일반 60%, 희귀 30%, 에픽 8%, 전설 2%

### 메타 성장 (Meta-Progression)
- [ ] **영구 해금 시스템**: 캐릭터 / 시작 아이템 / 패시브 해금
- [ ] **런 간 화폐**: 별도 통화 설정 (메타 성장에만 사용)
- [ ] **잠금 해제 트리**: 의존성 그래프로 설계 (순환 없이)
- [ ] **초기 파워 크리프 방지**: 첫 10런은 메타 성장 없이도 클리어 가능해야 함

## 수치 템플릿

### 적 스탯 기본 공식
```
Enemy HP    = BaseHP × (1 + 0.15 × floor) × EliteMultiplier
Enemy DMG   = BaseDMG × (1 + 0.10 × floor)
EliteMultiplier = 일반: 1.0 / 엘리트: 2.5 / 보스: 5.0
```

### 아이템 밸런싱 기준
```
Tier별 기댓값 (DPS 기준):
Normal  = Player DPS × 0.05 ~ 0.10 (5~10% 상승)
Rare    = Player DPS × 0.10 ~ 0.20
Epic    = Player DPS × 0.20 ~ 0.40
Legendary = 빌드 변화 수준 (수치가 아닌 메커닉 변화)
```

## Edge Case (장르 특수)

1. **빌드 수렴**: 항상 같은 아이템 조합만 강함 → 아이템 풀 주기적 패치 + 저주 시스템
2. **런 초반 포기**: 시작 5분에 약한 아이템만 나오면 즉시 재시작 유도 → 시작 아이템 보장
3. **무한 런**: 클리어 후 계속 진행 가능 시 무한 스케일링 설계 필요
4. **RNG 분산**: 100% 운빨 빌드 vs 실력 빌드 비율 권장: 실력 60% / 운 40%

## 레퍼런스 게임
- **Hades 2**: 메타 성장 + 내러티브 통합의 정석
- **Slay the Spire**: 카드 덱빌딩 + 시너지 설계 최고 사례
- **Vampire Survivors**: 극단적 캐주얼화, 자동 전투의 성공 사례
- **Dead Cells**: 액션 정밀도 + 빌드 다양성 균형
